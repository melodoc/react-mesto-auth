## Установка

``` shell
npm install
```

## Запуск

``` shell
npm start
```

Приложение «Место» загрузится на http://localhost:3000.

## Сборка приложения

``` shell
npm run build
```

Это создаст папку dist со скомпилированными и свернутыми компонентами.
